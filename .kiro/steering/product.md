# Product Overview

This is a **Weekly Routine Checklist** web application designed to help users track their weekly habits and goals in a flexible, stress-free manner.

## Core Features

- **Multi-category habit tracking**: Spiritual practices, exercise, programming, English practice, and basic habits
- **Task management**: Additional work items with completion tracking
- **Weekly review system**: Reflection on what went well, what was challenging, and adjustments for next week
- **Progress visualization**: Real-time progress bar showing completion percentage
- **Data persistence**: Local storage for maintaining state across sessions
- **Export/Import**: JSON-based data backup and restore functionality

## Target Audience

Arabic-speaking users looking to maintain consistent weekly routines across multiple life areas without rigid scheduling pressure.

## Design Philosophy

- **Flexible and realistic**: No pressure or failure feelings
- **Bilingual interface**: Arabic primary with some English elements
- **Visual progress tracking**: Clear feedback on weekly completion
- **Reflection-focused**: Emphasis on learning and adjustment rather than perfection
